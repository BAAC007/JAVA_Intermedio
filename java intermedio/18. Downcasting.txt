# Conversión de Tipo:
Para las clases, existen dos tipos de conversión.

Conversión Ascendente:
Puedes convertir una instancia de una subclase a su superclase.

Considera el siguiente ejemplo, asumiendo que Gato es una subclase de Animal. 

Animal a = new Cat();

# Java convirtió automáticamente la variable de tipo Cat a tipo Animal.

# Downcasting
Convertir un objeto de una superclase a su subclase se llama downcasting.

Ejemplo:

Animal a = new Cat();
((Cat)a).makeSound();

Esto intentará convertir la variable a al tipo Cat y llamará su método makeSound().
