#La API de Java proporciona clases especiales para almacenar y manipular grupos de objetos.

Una de estas clases es la ArrayList. Los arrays estándar de Java son de longitud fija, lo que significa que después de que se crean, no pueden expandirse o encogerse.

Por otro lado, las ArrayLists se crean con un tamaño inicial, pero cuando este tamaño se excede, la colección se amplía automáticamente.

Cuando se eliminan objetos, el ArrayList puede disminuir de tamaño. Observa que la clase ArrayList está en el paquete java.util, por lo que es necesario importarlo antes de usarlo.

Crea un ArrayList como lo harías con cualquier objeto.

Ejemplo:

import java.util.ArrayList;
//...
ArrayList colors = new ArrayList();


#Puedes especificar opcionalmente una capacity y type de objetos que el ArrayList contendrá:


ArrayList<String> colors = new ArrayList<String>(10);


#El código anterior define un ArrayList de Strings con 10 como su tamaño inicial.


#La clase ArrayList proporciona una serie de métodos útiles para manipular sus objetos.
