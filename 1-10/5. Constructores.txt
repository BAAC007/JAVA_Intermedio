#Los constructores son métodos especiales invocados cuando se crea un objeto y se utilizan para inicializarlos. 

Un constructor puede ser utilizado para proporcionar valores iniciales para los atributos del objeto.

- El nombre de un constructor debe ser el mismo que el nombre de su clase.

- Un constructor no debe tener un tipo de retorno explícito.

ejemplo:

public class Vehicle {
  private String color;
  Vehicle() {
     color = "Red";
  }
}

#El método Vehicle() es el constructor de nuestra clase, por lo tanto, siempre que se crea un objeto de esa clase, el atributo color se establecerá en "Rojo".

Un constructor también puede tomar parámetros para inicializar atributos.

ejemplo:

public class Vehicle {
  private String color;
  Vehicle(String c) {
    color = c;
  }
}

#El constructor es llamado cuando creas un objeto usando la palabra clave new .

Ejemplo:

public class MyClass {
  public static void main(String[ ] args) {
    Vehicle v = new Vehicle("Blue");
  }
}

#Una única clase puede tener múltiples constructores con diferentes números de parámetros.

Los métodos setter dentro de los constructores pueden ser usados para establecer los valores de los atributos.

ejemplo:

public class Vehicle {
  private String color;

  Vehicle() {
    this.setColor("Red");
  }
  Vehicle(String c) {
    this.setColor(c);
  }

  // Setter
  public void setColor(String c) {
    this.color = c;
  }
}

#La clase anterior tiene dos constructores, uno sin ningún parámetro que establece el atributo de color a un valor predeterminado de "Rojo", y otro constructor que acepta un parámetro y lo asigna al atributo.

Ahora, podemos usar los constructores para crear objetos de nuestra clase.

ejemplo:

public class Vehicle{
	private String color;

	Vehicle(){
		this.setColor("Red");
	}
	Vehicle(String c){
		this.color = c;
	}
	
	//Setter
	public void setColor(String c){
		this.color = c;
	}

	//Getter
	public String getColor(){
		return color;
	}
}

public class Program{
	public static void main(String[] args){
		//color will be "Red"
		Vehicle v1 = new Vehicle();
		
		//color will be "green"
		Vehicle v2 = new Vehicle("Green");
		
		System.out.println(v2.getColor());
	}
}
