#Java admite anidar clases; una clase puede ser miembro de otra clase.

Crear una clase interna es bastante simple. Solo escribe una clase dentro de otra clase. A diferencia de una clase, una clase interna puede ser privada. Una vez que declares una clase interna privada, no se puede acceder a ella desde un objeto fuera de la clase.

Ejemplo:

class Robot{
	int id;
	Robot(int i){
		id = i;
		Brain B = new Brain();
		B.think();
	}
	
	private class Brain{
		public void think(){
			System.out.println(id + "is thinking");
		}
	}
}

public class Program{
	public static void main(String[] arg){
		Robot R = new Robot(1);
	}
}
